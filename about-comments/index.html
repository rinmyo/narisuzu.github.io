關於敝站

敝站原名問題屋，爲數位化習題集，現爲鳴鈴的 weblog。 驅動自 Hugo, 左列所有功能完全基於 Github 部署, 不使用任何第三方伺服器与資料庫.

  • blog 程式 Hugo
  • 持續整合 Github Actions
  • 靜態頁面 Github Pages
  • 留言 💎 giscus

關於敝站的構建流程，可見於此。

格式

敝站使用十分高效的 $\KaTeX$ 作爲 $\LaTeX$ 公式渲染器. 中文排版基本按照中文文案排版指北的規範進行.
